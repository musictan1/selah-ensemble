# 셀라 합창단 웹사이트 이메일 인증 설정 가이드

## 이메일 전송 기능 설정 방법 (Gmail 예시)

1. Gmail 계정 준비
   - selahensemble1@gmail.com 계정 또는 전용 이메일 계정을 준비하세요
   - 관리용 계정을 새로 만드는 것을 권장합니다

2. 2단계 인증 활성화
   - Gmail 계정에 로그인 후 [계정 관리] -> [보안] 메뉴로 이동하세요
   - 2단계 인증을 활성화하세요

3. 앱 비밀번호 생성
   - 2단계 인증을 활성화한 다음, [앱 비밀번호] 메뉴로 이동하세요
   - "앱 선택"에서 "메일"을 선택하세요
   - "기기 선택"에서 "기타"를 선택하고 "셀라합창단" 등 식별 가능한 이름을 입력하세요
   - 생성된 16자리 앱 비밀번호를 복사하세요

4. 코드 수정
   - app.py 파일에서 send_email 함수를 찾으세요
   - smtp_username = "발신용 이메일 주소"
   - smtp_password = "생성한 앱 비밀번호"
   - 위 두 값을 수정하세요

5. 테스트
   - 수정 후 서버를 재시작하고 비밀번호 찾기 기능을 테스트하세요
   - 지정한 이메일로 인증번호가 발송되는지 확인하세요

## 문제 해결

1. "SMTPAuthenticationError" 오류가 발생하는 경우:
   - 이메일 주소와 앱 비밀번호가 정확한지 확인하세요
   - Gmail 보안 설정에서 "보안 수준이 낮은 앱의 액세스" 허용을 확인하세요

2. "인증 코드 전송 중 오류가 발생했습니다" 메시지가 표시되는 경우:
   - 서버 콘솔 로그를 확인하여 정확한 오류 메시지를 파악하세요
   - 인터넷 연결 상태를 확인하세요

3. 이메일은 발송되지만 수신되지 않는 경우:
   - 스팸 메일함을 확인하세요
   - Gmail 보안 설정에서 차단된 계정이 있는지 확인하세요

## 보안 주의사항

- 앱 비밀번호는 외부에 노출되지 않도록 주의하세요
- 정기적으로 앱 비밀번호를 변경하는 것이 좋습니다
- 이메일 전송 시 개인정보 보호에 유의하세요
